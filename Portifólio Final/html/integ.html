    Catapulta de pinguins - Brinquedo Natureza do Design

    Neste projeto, tinhamos o objetivo de fazer um produto (brinquedo) para crianças na faixa etária de 6 à 8 anos.

    O produto deveria ser bioinspirado, para ajudas as crianças a desenvolver certa empatia por animais.

    Para o brinquedo de Natureza do Design, foi feita uma catapulta que atirava pinguins feitos de bolinhas de ping-pong. A brincadeira consistia nas crianças pegando os pinguins e carregando a catapulta com eles para que eles pudessem saber como era voar. Depois de arremessá-los, as bolinhas firariam pulando de um lado para o outro enquanto as crianças tentam pegá-las.

    Projeto 3 - Mod Sim - Care Package

    Nesse projeto foi proposta a modelagem de uma caixa de suprimentos (care package) sendo entregue caindo de paraquedas para soldados. Para tornar o modelo mais real foram equacionados elementos como a resistência do ar, vento e a abertura do paraquedas a partir de certa altura.

    Por fim, foi o sistema foi modelado simultaneamente em diversas condições para demonstrar sob quais condições os suprimentos chegariam aos soldados.
